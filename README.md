## 📌 Apa itu Data Science?

Data Science adalah bidang yang menggabungkan:

- 📊 Statistik & Matematika  
- 💻 Programming  
- 🧠 Machine Learning  
- 📈 Data Visualization  
- 🗄️ Database & Data Processing  

Tujuannya adalah mengolah data menjadi insight untuk pengambilan keputusan.
